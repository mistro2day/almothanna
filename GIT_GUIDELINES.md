# إرشادات الالتزام بـ Git (Git Commit Guidelines)

> [!IMPORTANT]
> **قاعدة إلزامية لجميع المطورين والمساعدين الذكيين (AI Agents):**
> يجب كتابة جميع رسائل الالتزام (Commit Messages) باللغة العربية فقط في هذا المشروع وفي جميع المشاريع التابعة. يمنع استخدام اللغة الإنجليزية في رسائل الالتزام نهائياً.

## صيغة رسائل الالتزام المعتمدة:

يجب أن تصف رسالة الالتزام التغييرات المطبقة بوضوح واختصار باللغة العربية كالتالي:

* **إضافة ميزة جديدة**: `ميزة: [وصف الميزة باللغة العربية]`
  * مثال: `ميزة: إضافة فلترة المبيعات بين تاريخين وتصدير التقارير`
* **إصلاح خلل أو علة**: `إصلاح: [وصف الخلل المصلح]`
  * مثال: `إصلاح: حل مشكلة النقر المزدوج على أزرار الدفع`
* **تحسين وتجميل الواجهة**: `تحسين: [وصف التحسين البصري]`
  * مثال: `تحسين: تعديل انحناء زوايا الشريط الأخضر للنافذة المنبثقة`
* **تحديث إعدادات أو هيكلية**: `تحديث: [وصف التحديث هيكلي أو برمجياً]`
  * مثال: `تحديث: إضافة مسار حذف الفواتير في الواجهة الخلفية`
